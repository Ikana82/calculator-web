# Modern iOS-Style Calculator Web App

[![Live Demo](https://img.shields.io/badge/demo-live-pink.svg?style=for-the-badge)](https://ikana82.github.io/calculator-web/)

Sebuah aplikasi kalkulator berbasis web yang dirancang dengan estetika modern terinspirasi dari UI iOS/iPhone. Proyek ini menonjolkan kebersihan desain dan fungsionalitas.
